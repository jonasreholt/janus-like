About the Janus Playground

This playground is heavily based on the work of Michael Budde and Claus Skou Nielsen.

It let's you try out the Japa programming language directly from your browser, without having to install anything locally.

Code written in the textarea can be compiled into C++ by pressing the button C++.

Japa is based on the language Janus, and it's documentation can be found here.
